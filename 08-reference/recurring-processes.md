# Recurring Processes

## Processing New Documents

See `PROCESS.md`.

## Recording A Payment

1. Create and index the source using `PROCESS.md`.
2. Add the payment to `04-finance/ledger.md`.
3. Update the related project or finance file.
4. Add any follow-up to `02-dashboard/next-actions.md`.

## Recording A Quote

1. Create and index the source using `PROCESS.md`.
2. Add the contractor to `05-people/contractors.md` if missing.
3. Add the quote to the relevant project file.
4. Add the amount to the relevant finance file.
5. Add a recommendation if acceptance requires judgment.

## Recording A Decision

1. Create, reference, or index the source using `PROCESS.md`.
2. Add the final choice to `07-decisions/decision-log.md`.
3. Move any superseded option to `07-decisions/rejected-options.md`.
4. Update affected tasks, costs, and timeline entries.
