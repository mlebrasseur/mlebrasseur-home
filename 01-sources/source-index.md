# Source Index

Canonical registry for processed files, extraction notes, and chat-derived sources.

Use `SRC-YYYY-NNN` IDs.

For file sources, `Archived / Source Path` points to the moved original file and `Source Note` points to the extraction note. For chat sources, use `N/A` for `Archived / Source Path` and point `Source Note` to the chat note.
