# Inbox

This folder is for unprocessed inputs.

## Files

Drop new PDFs, images, receipts, contracts, quotes, emails, exports, or scans into `00-inbox/files/`.

Anything in `00-inbox/files/` should be considered not yet processed.

## Chat Notes

Use `chat-notes.md` only as a temporary holding area if context needs to be written down before it is converted into a proper source note under `01-sources/chat/YYYY/`.
